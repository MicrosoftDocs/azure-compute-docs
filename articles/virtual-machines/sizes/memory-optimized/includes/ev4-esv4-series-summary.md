# Customer intent: "As an IT infrastructure planner, I want to understand the specifications of the Ev4 and Esv4-series virtual machines, so that I can assess their suitability for memory-intensive enterprise applications."
---
The Ev4 and Esv4-series run on Intel® Xeon® Platinum 8473C (Sapphire Rapids), Intel® Xeon® Platinum 8370C (Ice Lake), or Intel® Xeon® Platinum 8272CL (Cascade Lake) processors in a hyper-threaded configuration, are ideal for various memory-intensive enterprise applications and feature up to 504GiB of RAM. It features an all core Turbo clock speed of 3.4 GHz.
